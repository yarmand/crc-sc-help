## Outlook (Email & Calendar)

**What it's for**: Sending and receiving CRC-SC email, managing your calendar, and scheduling meetings.

- Access at: [https://outlook.office.com](https://outlook.office.com)
- Use your `[REPLACE: @crc-sc.org]` address for all board-related communication.

## Microsoft Teams

**What it's for**: Real-time chat, video meetings, and file sharing with other board members.

- Access at: [https://teams.microsoft.com](https://teams.microsoft.com)
- See the [Teams guide](teams) for detailed instructions.

## SharePoint / OneDrive

**What it's for**: Storing and organizing board documents. SharePoint is for shared team files; OneDrive is for your personal files.

- SharePoint site: `[REPLACE: e.g. https://crc-sc.sharepoint.com/sites/Board]`
- OneDrive: [https://onedrive.com](https://onedrive.com)
- See the [Board Document Library guide](board-document-library) for details.

## Microsoft Loop

**What it's for**: Collaborative workspaces for brainstorming, meeting notes, and tracking tasks in real time.

- Access at: [https://loop.microsoft.com](https://loop.microsoft.com)
- See the [Loop Workspace guide](loop-workspace) for details.

## Word, Excel, PowerPoint (Online)

**What it's for**: Creating and editing documents, spreadsheets, and presentations â€” right in the browser.

- Access from the [Microsoft 365 home page](https://www.office.com) or open files directly from SharePoint/OneDrive.
- Multiple people can edit the same file at once (real-time co-authoring).

## Microsoft Forms

**What it's for**: Creating surveys, polls, and sign-up forms.

- Access at: [https://forms.office.com](https://forms.office.com)
- Useful for board votes, event RSVPs, and feedback collection.

---

{: .note }
> Not sure which tool to use? Start with **Teams** for communication and **SharePoint** for documents. Those are the two you'll use most.
